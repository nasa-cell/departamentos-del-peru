# Proyecto: Dado de los Departamentos del Perú

Aplicación web sencilla para seleccionar de forma aleatoria un departamento peruano y mostrar su información.

## Instrucciones

1. Abre `index.html` en tu navegador.
2. Haz clic en el dado fijo en la esquina superior izquierda.
3. El dado girará y mostrará un departamento al azar.
4. Repite las veces que desees.

## Archivos

- `index.html`: Interfaz principal.
- `styles.css`: Estilos y animaciones en 3D.
- `script.js`: Lógica del dado y datos de los departamentos.

> No hay sistema de puntaje, niveles, temporizadores ni derrotas. Solo educación y exploración.
